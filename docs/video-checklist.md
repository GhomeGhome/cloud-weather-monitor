# Video Checklist (3-10 min)

## Content flow
1. Team intro with faces on screen.
2. Problem statement and value proposition.
3. Live device demo:
   - indoor/outdoor/weather/time
   - motion and alert behavior
   - speech interactions
4. Cloud dashboard demo:
   - realtime panel
   - historical trends
   - events timeline
5. Reliability demo:
   - reboot sync from BigQuery
   - Wi-Fi configuration screen
6. Closing and GitHub link.

## Must-follow constraints
- Show no source code in the video.
- Human voice-over only.
- Keep pacing dynamic and practical.
